# 따사룸 웹사이트

프리미엄 유아ㆍ반려견 매트 전문 브랜드 "따사룸"의 공식 웹사이트입니다.

## 🎨 디자인 특징

- **현대적이고 깔끔한 디자인**: 미니멀한 레이아웃과 아이보리/베이지 톤
- **브랜드 컬러**: 따뜻한 아이보리/베이지 톤 (#8B7355, #F5F0EB, #E8E2D8)
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 모든 기기에서 최적화
- **스냅 스크롤**: 부드러운 섹션 전환 효과
- **견적계산기**: 실시간 가격 계산 기능

## 📁 프로젝트 구조

```
ddasaroom website 2025/
├── index.html                    # 메인 HTML 파일
├── pages/                        # 추가 페이지들
│   ├── products.html             # 제품 소개 페이지
│   ├── calculator.html           # 견적계산기 페이지
│   └── contact.html              # 고객센터 페이지 (예정)
├── assets/                       # 정적 자원들
│   ├── css/
│   │   └── styles.css            # CSS 스타일시트
│   ├── js/
│   │   ├── script.js             # 메인 JavaScript
│   │   └── calculator.js         # 견적계산기 JavaScript
│   └── images/
│       └── hero.png              # 히어로 배경 이미지
└── README.md                     # 프로젝트 설명서
```

## 🚀 주요 기능

### 네비게이션
- 반응형 햄버거 메뉴
- 드롭다운 서브메뉴
- 스크롤 시 배경 투명도 변경
- 부드러운 애니메이션 효과

### 히어로 섹션
- 풀스크린 레이아웃
- 그라데이션 배경과 오버레이
- 스크롤 인디케이터 애니메이션
- 반응형 텍스트 크기 조정

### 브랜드 소개 섹션
- 3열 레이아웃 (좌측: 브랜드 정보, 중앙: 로고, 우측: 브랜드명)
- 원형 로고 플레이스홀더
- 세로 텍스트 레이아웃

### 플로팅 버튼
- 우측 고정 위치
- 스토어 이동 및 상담 버튼
- 스크롤 시 나타나는 효과

### 견적계산기
- 실시간 가격 계산 기능
- 매트 종류별 가격 안내표
- 반응형 계산 결과 표시

### 추가 페이지
- 제품 상세 소개 페이지
- 견적계산기 전용 페이지
- 통합 네비게이션 시스템

## 💻 기술 스택

- **HTML5**: 시맨틱 마크업
- **CSS3**: Flexbox, Grid, 애니메이션, 반응형 디자인
- **JavaScript (ES6+)**: DOM 조작, 이벤트 처리, 성능 최적화
- **Google Fonts**: Noto Sans KR 폰트

## 🛠️ 설치 및 실행

1. 프로젝트 파일을 다운로드합니다.
2. 웹 서버에 파일을 업로드합니다.
3. `index.html` 파일을 브라우저에서 열어 확인합니다.

### 로컬 개발 서버 실행 (선택사항)

```bash
# Python 3가 설치된 경우
python -m http.server 8000

# Node.js가 설치된 경우
npx serve .

# VS Code Live Server 확장 사용
```

## 🎨 커스터마이징

### 색상 변경
`styles.css` 파일에서 CSS 변수를 수정하여 브랜드 색상을 변경할 수 있습니다:

```css
/* 주요 브랜드 컬러 */
--primary-pink: #ffb3d1;
--secondary-pink: #ff9ec7;
--accent-pink: #ff6b9d;
```

### 폰트 변경
Google Fonts에서 다른 폰트를 선택하고 `index.html`의 링크를 업데이트하세요.

### 콘텐츠 수정
`index.html` 파일에서 텍스트 내용을 직접 수정할 수 있습니다.
