# Estimator

웹 기반 공간 매트 배치 도구입니다. `index.html`을 열면 브라우저에서 바로 프로젝트를 확인할 수 있습니다.

## 작업 커밋 & 푸시

1. 수정 사항 확인
   ```bash
   git status -sb
   git diff
   ```
2. 변경 파일 스테이징
   ```bash
   git add <파일명>
   # 전부 추가하려면
   git add .
   ```
3. 커밋 생성
   ```bash
   git commit -m "변경 내용을 설명하는 메시지"
   ```
4. 원격 저장소로 푸시
   ```bash
   git push
   ```

필요에 따라 `git log --oneline`으로 커밋 히스토리를 확인하거나 `git status`로 현재 상태를 다시 확인하세요.
